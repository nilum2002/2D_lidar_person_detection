# DR-SPAAM ROS 2 Implementation

This repository provides a ROS 2 implementation of the **DR-SPAAM** framework for 2D LiDAR-based person detection.

The implementation is based on the original DR-SPAAM paper:
- **Paper:** https://arxiv.org/abs/2004.14079

The original implementation can be found here:
- **GitHub:** https://github.com/VisualComputingInstitute/DR-SPAAM-Detector

All credit for the DR-SPAAM algorithm goes to the original authors. This repository focuses on providing a ROS 2 implementation and integration of their work.
